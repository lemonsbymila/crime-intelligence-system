# Crime Intelligence Case Logging System

## Overview
This project is a Python-based case logging system that simulates how crime or cybersecurity incident reports are recorded and structured.

It allows users to input case details, generates a formatted report, and stores data using file handling.

---

## Features
- Interactive case input system
- Structured report generation
- File storage of case logs
- Simple console-based interface

---

## Technologies Used
- Python 3
- Input/output handling
- File handling (write operations)
- Variables and basic control flow

---

## What I Learned
- How to collect and store user input
- How to structure data using variables
- How to generate formatted output
- How to save data to external files
- Basic simulation of real-world logging systems
